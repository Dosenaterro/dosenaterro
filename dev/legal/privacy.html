Politique de confidentialité

Dernière mise à jour : 1er janvier 2024

1. Introduction

La présente politique de confidentialité décrit comment Real Dosenaterro (« nous », « notre », « nos ») collecte, utilise et protège les informations personnelles que vous (« l'utilisateur », « vous », « votre ») nous fournissez via notre site web.

Nous nous engageons à protéger votre vie privée et à assurer la sécurité de vos données personnelles conformément aux lois en vigueur en Côte d'Ivoire.

2. Données collectées

Nous collectons les types de données suivants :

2.1 Données fournies volontairement

Lorsque vous utilisez notre formulaire de contact, nous collectons :

  • Votre nom et prénom
  • Votre adresse email
  • Votre numéro de téléphone (optionnel)
  • Le type de projet souhaité
  • Votre budget estimé (optionnel)
  • Le contenu de votre message

2.2 Données collectées automatiquement

Lors de votre navigation sur notre site, nous collectons automatiquement :

  • Votre adresse IP
  • Le type de navigateur utilisé
  • Le système d'exploitation
  • Les pages visitées et le temps passé sur chaque page

3. Finalités du traitement

Vos données personnelles sont collectées pour les finalités suivantes :

  • Répondre à vos demandes — Utilisation de vos coordonnées pour vous recontacter
  • Améliorer nos services — Analyse des données de navigation
  • Sécurité — Prévention de la fraude et sécurisation de notre infrastructure

4. Durée de conservation

Vos données sont conservées pendant les durées suivantes :

  • Données de contact — 2 ans à compter du dernier contact
  • Données de navigation — 12 mois maximum

5. Vos droits

Vous disposez des droits suivants concernant vos données personnelles :

  • Droit d'accès — Obtenir une copie de vos données
  • Droit de rectification — Corriger des données inexactes
  • Droit à l'effacement — Demander la suppression de vos données
  • Droit d'opposition — Vous opposer au traitement de vos données

Pour exercer ces droits, contactez-nous à : dosenaterroa@gmail.com
